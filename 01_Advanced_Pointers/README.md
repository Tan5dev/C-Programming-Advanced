📂 01_Advanced_Pointers

🎯 Objective

Learn how pointer arithmetic works in C and how pointers can be used to traverse arrays efficiently.

📚 Concepts Covered

- What is Pointer Arithmetic?
- Incrementing Pointers
- Traversing Arrays using Pointers
- Difference between "arr[i]" and "*(arr+i)"
- Memory Addresses

🧠 Why Learn This?

Pointers are one of the most powerful features of C. Understanding pointer arithmetic is essential before learning:

- Dynamic Memory Allocation
- Linked Lists
- Stacks
- Queues
- Trees
- Advanced Data Structures

🚀 Expected Output

The program will:

1. Store numbers in an array
2. Traverse the array using pointers
3. Print values and memory addresses
4. Demonstrate pointer arithmetic

📖 Key Formula

arr[i]

is equivalent to

*(arr + i)

Both access the same element in memory.

🎯 Challenge

Modify the program to:

- Find the largest number using pointers
- Find the smallest number using pointers
- Calculate the average using pointers
